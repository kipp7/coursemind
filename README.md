# CourseMind

CourseMind is a school-facing course agent platform. The first milestone is a Web MVP for demonstrating course-aware Q&A, citations, teacher oversight, and a RAG-first architecture.

## Current MVP

Open the static prototype:

```text
index.html
```

The current prototype demonstrates:

- student, teacher, and admin role surfaces
- course switching
- cited answer experience
- knowledge base status
- architecture path for Dify / RAGFlow / model APIs
