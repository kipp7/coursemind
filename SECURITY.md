# Security And Public Repository Policy

CourseMind is currently an MVP-stage public repository. Do not commit:

- API keys, tokens, credentials, or private endpoints
- student private data or learning records
- private school course materials
- local databases, logs, cache files, or generated runtime state
- copyrighted teaching materials unless publication rights are explicit

## Reporting

If you find a security issue in this repository, open a private report through GitHub security advisories when available, or contact the maintainer directly through the repository owner account.

## Data Handling

The MVP uses sample content only. Any future real deployment should keep course materials, student questions, teacher corrections, and model traces in protected storage with explicit access controls.
